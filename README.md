# TraceMind AI

AI-powered DevOps Incident Analysis Platform built with the MERN Stack and OpenAI.

## 🚀 Overview

TraceMind AI helps developers analyze application incidents using AI.

### Features

- AI-powered incident analysis
- Root cause detection
- Log management
- Incident dashboard
- Authentication
- File uploads
- Analytics

## 🛠 Tech Stack

### Frontend

- React
- Vite
- React Router
- Axios

### Backend

- Node.js
- Express.js

### Database

- MongoDB

### AI

- OpenAI API
